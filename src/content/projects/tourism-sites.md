## Site Features

Each tourism site includes:
- Detailed destination information and highlights
- Interactive Mapbox maps showing location and nearby attractions
- Cards linking to official ticketing platforms
- Mobile-responsive design optimized for mobile bookers

## Technical Implementation

Each site is optimized for search engines and performance:

- **Schema.org Structured Data**: LocalBusiness, Event, and Breadcrumb schemas with JSON-LD
- **On-Page SEO**: Descriptive meta titles/descriptions, proper heading hierarchy, alt text, internal linking
- **Performance**: Minified assets, optimized images, lazy loading, Cloudflare CDN, Firebase hosting
- **Conversion**: Clear CTAs linking to ticketing, trust signals, mobile-optimized booking flow

## Technologies Used

- Frontend: HTML5, CSS3, Vanilla JavaScript
- Maps: Mapbox GL JS
- Hosting: Firebase, Cloudflare
- SEO: Technical SEO, Schema.org, Open Graph
